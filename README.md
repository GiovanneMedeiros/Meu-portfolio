# Projetos

## File Converter Web

Aplicação web desenvolvida em React que permite converter arquivos diretamente pelo navegador através de uma API de conversão.

Principais funcionalidades:

* Upload de arquivos
* Conversão entre formatos
* Download do arquivo convertido
* Interface moderna e responsiva

Tecnologias utilizadas:

React • JavaScript • HTML • CSS • REST API

Demo
https://file-conversion-api.vercel.app

---

## File Conversion API

API responsável pelo processamento e conversão de arquivos utilizada pelo File Converter Web.

O sistema utiliza arquitetura modular e processamento assíncrono para lidar com tarefas de conversão de forma eficiente.

Principais funcionalidades:

* Conversão de arquivos entre diferentes formatos
* Processamento em background
* Arquitetura modular
* Estrutura REST para integração com frontend

Tecnologias utilizadas:

Node.js • Express • Redis • REST API

GitHub
https://github.com/GiovanneMedeiros/file-conversion-api

---

## AuthFlow API

API de autenticação desenvolvida para gerenciar cadastro de usuários, login e controle de acesso utilizando JSON Web Tokens (JWT).

Principais funcionalidades:

* Cadastro de usuários
* Login com autenticação JWT
* Geração e validação de tokens
* Rotas protegidas
* Middleware de autenticação

Tecnologias utilizadas:

Node.js • Express • JWT • REST API

# Executar o Projeto Localmente

Clone o repositório:

git clone https://github.com/GiovanneMedeiros/Meu-portfolio.git

Entre na pasta do projeto:

cd Meu-portfolio

Instale as dependências:

npm install

Execute o projeto:

npm run dev

Abra no navegador:

http://localhost:5173
