loja de carros

MotorHub é um site de classificados de carros, onde os usuários podem buscar e visualizar veículos. O site é moderno, minimalista e responsivo, usando o Tailwind CSS para o design e JavaScript para a lógica de exibição dos carros. O objetivo principal é proporcionar uma experiência de navegação simples e agradável para quem deseja comprar ou vender um veículo online.
